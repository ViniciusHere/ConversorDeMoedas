<h2 align="center"> Conversor de moedas </h2>

## Tecnologias
- Java

## About
<p>Este programa realiza requisições para uma API de taxas de câmbio, permitindo que você execute conversões monetárias entre diferentes moedas de forma simples e eficiente. 
   Com ele, você pode consultar as taxas de câmbio mais recentes e converter qualquer valor entre moedas, de acordo com as suas necessidades. 
   Basta inserir as moedas de origem e destino, e o programa retornará o valor convertido com base nas taxas atualizadas fornecidas pela API.
</p>
